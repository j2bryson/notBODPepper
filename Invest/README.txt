Pepper-JackPosh contains the code necessary to compile and run the game using the social robot Pepper, and should be installed and run according to the steps found at https://qisdk.softbankrobotics.com/sdk/doc/pepper-sdk/ch1_gettingstarted/running_application.html
The files created for this project are:
-InvestmentBehaviourLibrary.java
-investment_input.top
-investment_listen.top
-investment_listen_na.top
-investment_rules.top
-investment_rules_na.top
-plan_investment.xml
The BasicBehaviourLibrary.java was also rewritten slightly, with comments highlighting what was changed


GUI contains all of the code necessary to run the standalone GUI to accompany the game
